OK Downloader
Developer: Arif Kılıç

Methodology: Vibe Coding

Technical Specifications
OK Downloader is a desktop application developed for downloading videos and playlists exclusively from YouTube. The software includes an integrated media player for audio files. The application supports English, Turkish, German, Russian, and Spanish language options.

Additional Technical Details
Language: Python 3.13

UI Framework: CustomTkinter (Modernized Tkinter interface)

Core Libraries: * yt-dlp: Used for handling YouTube video and playlist extraction.

pygame: Powers the internal audio player and playback controls.

Pillow (PIL): Handles image processing for UI elements.

Packaging: Bundled into a standalone executable using PyInstaller.

Security and Privacy
Data Privacy: This application does not collect, store, or transmit any personal user data. All download processes are handled directly between the user's machine and YouTube servers.

Network Security: Connections are established via secure HTTPS protocols.

Code Integrity: The software is digitally signed by Arif Kılıç. The authenticity can be verified under the "Digital Signatures" tab in the file properties.

Antivirus Notice: Due to the PyInstaller bundling process, some antivirus engines may trigger a "False Positive" alert. This is a technical characteristic of Python-based executables. The software contains no malicious code.

Usage
Download the executable from the Releases section.

Enter the YouTube video or playlist URL into the application.

Access the internal audio player to manage downloaded files.

If prompted by Windows SmartScreen, select "More Info" and "Run Anyway."
